#TinyApp Project

##Final Product
!!![screenshots]

##Dependancies:

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- method-override

## Getting Started

-Install dependancies ( `npm install` )
- Run the development web server ( `node express_server.js )
